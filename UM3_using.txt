Emulator UM3

Readme по аргументам командной строки

--run, -c, --txt-bin, --bin-txt, --help
перекрывают друг друга, вместе не использовать

--run
Необходим чтобы запустить "программу" УМ3 на выполнение

--punched=<file_name>
Задает имя файла, содержащего "программу" УМ3

-c
Опция, необходимая, если требуется только собрать бинарный файл
"программы" УМ3 из исходного файла

--txt-bin="file_name"
--bin-txt="file_name"
Конвертируют файл из бинарного вида в "текстово-бинарный" (см. примеры)
или наоборот, слева направо

--
